# Plant Disease Classifier

A MobileNetV2 implementation built with transfer learning in TensorFlow for Plant Disease classification.

## Concepts Implemented

- Transfer Learning
- Prefetch
- Eraly Stopping
- Model Checkpoint
- Preprocessing
- Data Augmentation

## Dataset

- PlantVillage
- 15 classes
- 224x224 RGB images
